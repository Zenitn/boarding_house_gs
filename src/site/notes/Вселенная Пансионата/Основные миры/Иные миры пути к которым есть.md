На данный момент перемещаться между мирами и планетами — дело не слишком сложное.  
В основном перемещение осуществляется золотыми мостами. Такие абстрактные строения есть на большинстве планет. Они представляют собой мост из золотых линий. Найдя такой мост, вы можете пройтись по нему и выбрать, куда хотите попасть.

Но есть нюансы. Не все миры сразу открыты. Некоторые придётся открывать разными действиями, а другие вообще не связаны мостами.  
То есть есть несколько планет, куда можно попасть лишь уникальными способами...


Открытые миры:
+ Живой мир
+ ГОРА
+ Затопленный мир
+ Мир БЕЗДНА
+ Филиал 
+ Фруктовый мир
+ Гнездовье
